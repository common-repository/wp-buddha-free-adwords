== Description ==
Write your google ads (adwords) in your wordpress page or post, Create UTM url for tracking on Google Analytics for FREE.

Features:
1. write google ads
2. **NEW** - write extanded google ads
3. UTM url generator
4. Ads backup

Created by [WP Buddha](https://wp-buddha.com/ \"wp buddha - online marketing solutions\") 


== Installation ==
How to install the plugin and get it working.
1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the \'Plugins\' screen in WordPress
